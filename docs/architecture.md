## 6. MГҙ HГ¬nh BбәЈo Mбәӯt

```text
Request вҶ’ Rate Limit вҶ’ JWT Verify вҶ’ Role Check вҶ’ Handler вҶ’ Response
              в”Ӯ             в”Ӯ            в”Ӯ
              в–ј             в–ј            в–ј
         429 Too Many    401         403 Forbidden
         Requests       Unauthorized
```

- **Rate Limit**: IP + email + endpoint, cб»ӯa sб»• trЖ°б»Јt 60s
- **JWT**: HS256, thб»қi hбәЎn configurable, cГі JTI chб»‘ng replay
- **RBAC**: 3 role (patient, doctor, admin), kiб»ғm tra trГӘn tб»«ng endpoint
- **IoT Auth**: Token riГӘng tб»«ng thiбәҝt bб»Ӣ, hash trong DB, MAC mapping
